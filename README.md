<h2>Setting Function:<br></h2>
<p><pre>
  Speed up      Page up
  Speed down    Page down
  Robot x+      →
  Robot x-      ←
  Robot y+      ↑
  Robot y-      ↓
  Show Ear      F1
  Eye Light     F2
  Change Skin   F3
</pre></p>

<h2>Adjust Function:<br></h2>
<p><pre>
  Float Dress   Z
  Head Nod      X
  Head Shake    C
  Rotate Ear    V
  I Watch You   B
</pre></p>

<h2>Posture Function:<br></h2>
<p><pre>
  Idle          Space
  Pose          A
  Sit           S
</pre></p>

<h2>Action Function:<br></h2>
<p><pre>
  Wave hand     Q
  Walk          W
  Jump          E
</pre></p>

<h2>Series Action Function:<br></h2>
<p><pre>
  Fly           0
  Dance         1
  Space walk    2
  Shy           3
  Rolling       4
  Big Wheel     5
</pre></p>
